## CRUD API
CRUD RESTful API made with Go and SQlite

## Getting started
To get started, run the following command:

```bash
# Start the project
go run ./
```

## Usage
### `/api/msg`
### GET
```
example.com/api/msg
```
### POST
```json
{
  "username": "",
  "msg": ""
}
```
### DELETE
```json
{
  "id": "",
}
```
### PATCH
```json
{
  "id": "",
  "msg": ""
}
```
